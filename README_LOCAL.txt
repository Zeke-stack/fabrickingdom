# Local Minecraft Kingdom Server Setup

## Files Downloaded:
- ✅ paper.jar (PaperMC 1.21.1-133)
- ✅ eula.txt (EULA accepted)
- ✅ server.properties (Kingdom configuration)
- ✅ plugins/KingdomCommands-1.0.0.jar (Kingdom plugin)
- ✅ start.bat (Startup script)

## To Run Locally:
1. Double-click `start.bat`
2. Server will start on port 25565
3. Connect with Minecraft client to localhost:25565

## Server Configuration:
- Port: 25565
- Difficulty: Normal
- Gamemode: Survival
- MOTD: Kingdom Server - Medieval Roleplay
- RCON enabled on port 25575

## Plugin Features:
- Kingdom-themed commands (/kban, /kkick, /kop, /kdeop)
- Coin economy (/coins, /balance)
- Medieval UI and messages

## For Railway Deployment:
Copy all files in this directory to your Railway project root.
